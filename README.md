# Prestige Contractors Pvt Ltd — Official Website

Professional construction consultancy website built with React + Vite.

**Owner:** Gullsher Fareed  
**Business:** Prestige Contractors Pvt Ltd

## Pages
- `/` — Home (Hero, Services, Why Us, CTA)
- `/services` — All Services
- `/projects` — Project Portfolio
- `/about` — About & Team
- `/contact` — Contact Form

## Tech Stack
- React 18 + Vite
- React Router DOM
- Lucide React (icons)
- Pure CSS (no UI framework)

## Local Development
```bash
npm install
npm run dev
```

## Deploy to Vercel
```bash
# 1. Push to GitHub
git init
git add .
git commit -m "Initial commit — Prestige Contractors website"
git remote add origin https://github.com/YOUR_USERNAME/prestige-contractors.git
git push -u origin main

# 2. Go to vercel.com → New Project → Import your GitHub repo
# 3. Vercel auto-detects Vite — click Deploy
```
